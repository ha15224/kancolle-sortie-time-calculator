戦果や海域RTAに使える平均出撃時間の計算機
※注1:基本的に通常海域での使用を想定して作成しています．イベント海域に現れる特殊な敵（潜水棲姫，空母棲姫等）は一部対応していません．

使い方：
1. Sortie simのdevtools(F12)から「JSONダウンロード用コード」を実行し，戦闘ログAPIを取得します（数分かかります）．
   コードをそのままconsoleにコピペしてください

2. 取得したJSONを入力し，戦闘アニメーションの統計を取ります．
※注2: ボスに到達した出撃しか認識されません．撤退は無視されます
※注3: 表示されるAverage timeは戦闘アニメーションのみをカウントした秒数となります．羅針盤，マス移動，リザルト等にかかる時間はすべて無視されます

3. 出力された平均時間には中破演出とバケツ操作の時間が入っていません．得られる平均時間の値に8秒*[シミュから出されたバケツ使用率]を足してください（戦果周回でなくRTA目的なら8秒を5秒にする）．また，ダメコン使用時は8秒*[シミュから出されたダメコン使用率]も足してください．

使用データ：https://x.com/shiro_sh39/status/1562484362553110529

対応している仕様（2026/01/19時点）：
索敵（艦載機あり／なし）
航空戦の存在有無
対空CI発動
砲撃・連撃・弾着CI・対潜・空母砲撃・空母CI・開幕雷撃・閉幕雷撃等の動き
夜戦移行時の追加時間
夜間瑞雲・魚雷CI・主CI等一部の夜戦CI

実装待ちの仕様（2026/01/19時点）：
対地攻撃
ジェット爆撃
水爆による対潜攻撃
基地航空隊
連合艦隊
タッチ攻撃
煙幕発動
友軍使用
イベント海域
夜戦CIの正確なタイミング（データがない）
