戦果や海域RTAに使える平均出撃時間の計算機
※注1:通常海域のみ対応しています．イベントには使えません．
※注2:実行にはpythonを使います．使いたい方はまずインストールしてください．

実行手順
1. 本ツールをダウンロードしてください（https://github.com/ha15224/kancolle-sortie-time-calculator/archive/refs/tags/kancolle.zip）

2. Sortie simのdevtools(F12)から「download_sortie_api.js」を実行し，戦闘ログAPIを取得します（数分かかります）．コードをそのままconsoleにコピペしてください
  → 1万回出撃分のデータがダウンロードされます．（大きすぎてエラーが出る場合等は適宜調整してください）

3. ダウンロードされたjsonファイルをmain.pyと同じdirに置きます

4. main.pyで読み込むjsonファイルの名前を指定します
8行目： with open("[ここを修正].json", "r", encoding="utf-8") as f:

5. コード（main.py）を実行し，戦闘アニメーションの統計を取ります．pythonスクリプトを実行する方法は割愛します（チュートリアルを探してください）
※注3: ボスに到達した出撃しか認識されません．撤退は無視されます
※注4: 表示されるAverage timeは戦闘アニメーションのみをカウントした秒数となります．羅針盤，マス移動，リザルト等にかかる時間はすべて無視されます

6. 出力された平均時間の値に8秒*(シミュから出されたバケツ使用率，戦果周回でなくRTA目的なら8秒を5秒にする)を足します，ダメコン使用時は8秒*(シミュから出されたダメコン使用率)も足します
