3-4下時速比較用コード
※注1:対応海域は3-4下のみです．他の海域やルートには対応していません

実行にはpythonを使います（numpyも必要）．使いたい方はまずこれらをインストールしてください．

実行手順
1. Sortie simのconsole(F12)から「download_sortie_api.js」を実行し，戦闘ログAPIを取得します（数分かかります）

2. ダウンロードされたjsonファイルをmain.pyと同じdirに置きます

3. main.pyで読み込むjsonファイルの名前を指定します
5行目： with open("[ここを修正].json", "r", encoding="utf-8") as f:

4. コードを実行し，戦闘アニメーションの統計を取ります
※注2: ボスに到達した出撃しか認識されません．撤退は無視されます
※注3: 表示されるAverage timeは戦闘アニメーションのみをカウントした秒数となります．羅針盤，マス移動，リザルト等にかかる時間はすべて無視されます

5. 出力された平均時間の値に8秒*(シミュから出されたバケツ使用率)を足します，ダメコン使用時は8秒*(シミュから出されたダメコン使用率)も足します

変数の和訳：
detection_plane：艦載機あり索敵成功（の回数，下記同様）
detection_noplane：艦載機なし索敵成功
dogfight_single：航空戦1艦隊のみ参加
dogfight_double：航空戦2艦隊とも参加
AACI：対空CI
single：単発攻撃
DA：連撃
otorp：開幕雷撃
ctorp：閉幕雷撃
cvshell：空母の単発攻撃
yasen：夜戦突入
nzci：夜間瑞雲CI
gunci：主主主CI
※注4: 3-4に現れるヲ級flagshipは制空を取られていても空母CIが出せません．
