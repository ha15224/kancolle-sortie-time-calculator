戦果や海域RTAに使える平均出撃時間の計算機
※注1:通常海域のみ対応しています．イベントには使えません．
※注2:実行にはpythonを使います．使いたい方はまずインストールしてください．

実行手順
1. Sortie simのdevtools(F12)から「download_sortie_api.js」を実行し，戦闘ログAPIを取得します（数分かかります）．コードをそのままconsoleにコピペしてください

2. ダウンロードされたjsonファイルをmain.pyと同じdirに置きます

3. main.pyで読み込むjsonファイルの名前を指定します
8行目： with open("[ここを修正].json", "r", encoding="utf-8") as f:

4. コードを実行し，戦闘アニメーションの統計を取ります
※注3: ボスに到達した出撃しか認識されません．撤退は無視されます
※注4: 表示されるAverage timeは戦闘アニメーションのみをカウントした秒数となります．羅針盤，マス移動，リザルト等にかかる時間はすべて無視されます

5. 出力された平均時間の値に8秒*(シミュから出されたバケツ使用率)を足します，ダメコン使用時は8秒*(シミュから出されたダメコン使用率)も足します
